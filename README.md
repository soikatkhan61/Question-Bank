# Question-Bank

## Clone this repository

    git clone https://github.com/soikatkhan61/Question-Bank.git && cd Question-Bank
    
## Install all dependencies

    yarn

## Run frontend
  
    yarn run dev
    
## Run Backend

    yarn run server
